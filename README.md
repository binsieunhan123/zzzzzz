## Docker-Compose安装部署
```bash
version: '3.9'
services:
    vue-idle-xiuxian:
        image: kowming/vue-idle-xiuxian:latest
        container_name: xiuxian
        restart: unless-stopped
        ports:
            - 8183:8080
        tty: true
        stdin_open: true
```

## NPM部署
```bash
# 安装依赖
npm install

# 开发环境运行
npm run dev

# 构建生产版本
npm run build
```
